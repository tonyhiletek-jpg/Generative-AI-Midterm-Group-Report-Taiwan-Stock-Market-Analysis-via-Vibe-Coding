# Vibe Coding: 台股半導體、AI 與 IC 設計產業分析

## 專案簡介
本專案為「生成式 AI 期中報告」的實作產物，核心理念採用 **Vibe Coding** 開發模式。
我們透過「定義意圖」來指揮 AI 撰寫複雜的數據爬蟲與視覺化程式，將重心放在觀察市場的「感覺 (Vibe)」與趨勢判讀，而非傳統的底層語法撰寫。

## 開發環境
- **語言**: Python 3.10+
- **執行平台**: vscode / Jupyter Notebook
- **關鍵套件**:
  - `yfinance`: 抓取台灣證券交易所之每日歷史數據。
  - `mplfinance`: 專業金融級圖表庫，用於繪製符合台股標準（紅漲綠跌）的 K 線圖。
  - `pandas`: 用於時間序列數據的處理與對齊。

## 涵蓋族群
本報告針對以下三大關鍵產業族群進行分析（每類 5 檔，共 15 檔指標股）：
1. **半導體**: 包含台積電、聯電、日月光、旺宏等權值龍頭。
2. **AI 概念股**: 包含廣達、鴻海、緯創、緯穎、英業達伺服器與硬體代工大廠。
3. **IC 設計**: 包含聯發科、世芯-KY、聯詠、瑞昱、南電技術密集型指標股。

## AI 工具鏈與開發流程
- **意圖定義 (Prompt Engineering)**: 向 AI 描述所需的選股邏輯與視覺化規格。
- **自動化代碼生成**: 利用生成式 AI 產出完整的資料處理流與 `matplotlib` 繪圖邏輯。
- **趨勢判讀**: 利用 AI 對生成的圖表（均線排列、成交量縮放）進行深度學習判讀，產出最後的市場分析結論。

## 如何執行
1. 在環境中安裝必要依賴：`pip install yfinance mplfinance`
2. 執行 `stock_analysis.py`。
3. 系統將自動產出 15 張包含 5MA、20MA 與成交量的專業 K 線圖。
4. 參考程式末尾的結論區塊獲取 AI 分析建議。
